# AI Companion

A modern, responsive AI Companion web application built with Flask and Vanilla JavaScript.

## Features
- **Intelligent Chat Interface**: Real-time simulated AI conversations with "thinking" animations.
- - **Adaptive Learning Quiz**: Interactive assessment tool to track user progress.
  - - **Dynamic Dashboard**: Personalized insights and activity monitoring.
    - - **Responsive Design**: Fully optimized for desktop and mobile devices.
     
      - ## Tech Stack
      - - **Backend**: Flask (Python)
        - - **Frontend**: HTML5, CSS3 (Modern Flexbox/Grid), Vanilla JavaScript
          - - **Icons**: Lucide Icons
